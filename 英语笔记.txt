基本概念：
主语：句子的主体，引领了整个句子
谓语：谓语动词，用于主语动作状态或特征的陈述或说明，指出"做什么(what to do)" ， "是什么(what is this)"
宾语：谓语动词后面跟着的东西
系动词：全称linking verb，就是只起联系作用的动词。最常见的系动词就是be动词（比如am、is、are、was、were），一批感官动词可以当系动词（比如：feel感觉起来、sound听起来、look看起来）。示例：You are beautiful、You look beautiful

缩写字典：
S：subject
V：verb
O：object
C：completement

一、五大基本句型 // 一句话：我记得，我给你讲过一个故事。那个故事总能让你开心。故事只有六个字：你好美，我爱你。
主谓（SV） // I remember
主谓宾（SVO） // I love you
主谓宾宾（SVOO） // I told you a story，宾语可分为：直接宾语（a story。told这个动作直接作用的对象，说出来的不是你，而是故事）和间接宾语（you）
主谓宾宾补（SVOC） // The story made you happy，宾补（补充说明宾语的部分，补语可以是形容词，或者是形容词性的东西，也可以是名词）
主系表（SVC） // You are beautiful，系代表系动词，表的全称是表语（本质和宾补是一个东西，只不过宾补是补充说明宾语的，而表语是补充说明主语的。表语可以是形容词（You are beautiful），或者是形容词性的东西，也可以是名词（You are a beauty））

二、词法 // 各种不同词性的单词形式变化
名词 // 单复数
动词 // 时态，分为谓语动词、系动词等
冠词 // a、an、the
形容词 // 比较级、最高级
副词 // 比较级、最高级
介词
连词
代词

三、修饰成分 // 不影响句子主体意思的能翻译成“xxx的/地”的成分
形容词和副词，属于修饰成分
形容词和副词性质的从句，也属于修饰成分
